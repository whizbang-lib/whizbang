# Whizbang Scripts

## show-diagnostics.ps1

Shows all diagnostics generated by the Whizbang source generator.

### Usage

**Command Line:**
```powershell
pwsh scripts/show-diagnostics.ps1
```

### VS Code Integration

**Quick Access (No Setup):**
1. Press `Cmd+Shift+P` / `Ctrl+Shift+P`
2. Type: **"Tasks: Run Task"** (or just "run task")

### Prerequisites

**PowerShell Core (pwsh) is required** - [Install here](https://github.com/PowerShell/PowerShell#get-powershell)

- **Windows**: Included with Windows 10+, or install PowerShell Core 7+ for latest features
- **macOS**: `brew install powershell`
- **Linux**: See [installation instructions](https://learn.microsoft.com/en-us/powershell/scripting/install/installing-powershell-on-linux)

### Output

The script shows all WHIZ diagnostics including:
- **WHIZ001** (Info) - Discovered receptors (green text)
- **WHIZ002** (Warning) - No receptors found (yellow text)
- **WHIZ003** (Error) - Invalid receptor implementations (red text)
